# Beyond The Visa v16 — Phase 3–5 Prototype

This build adds a working front-end prototype for nurses and midwives:

- Community posts and likes
- Nursing and midwifery jobs with search and saved vacancies
- Anonymous employer reviews
- AI-style CV profile, supporting statement and interview coaching tools
- Free, Premium and Employer plan screens
- Admin dashboard and vacancy creation
- Local device storage so demo content persists after refresh

## Important
This is a front-end MVP. Before accepting real users, payments, employer reviews or confidential information, connect the app to a secure backend such as Firebase or Supabase and add moderation, authentication rules, privacy notices and server-side AI calls. Never place private API keys in GitHub Pages.
